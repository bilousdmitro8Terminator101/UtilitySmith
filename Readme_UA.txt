## Опис:

BridgeSmith — це повноцінна консольна програма, написана на мові командного рядка Windows (Batch). Вона працює поверх ядра CMD і надає користувачу набір системних утиліт та інструментів через власне меню. Програма створена для швидкого доступу до ключових функцій Windows: діагностики, очищення, керування живленням, запуску системних компонентів, та відновлення системи.

### Robocopy:

## Robocopy — Надійне копіювання файлів для Windows

**Robocopy** (Надійне копіювання файлів) — це вбудована утиліта Windows для надійного копіювання та синхронізації файлів і папок.
Вона набагато потужніша за стандартне копіювання через Провідник і підходить для роботи з великими обсягами даних.

### Основні характеристики

**Надійність** — автоматично повторює спроби при помилках читання/запису.
**Гнучкість** — десятки ключів для налаштування (фільтрація, дзеркалювання `/MIR`, багатопотоковість `/MT`).
**Журнали** — веде детальний журнал виконання (`/LOG:file.txt`).
**Синхронізація** — дзеркалювання папок, зберігаючи структуру та атрибути.
**Продуктивність** — працює швидше за стандартне копіювання в Провіднику.

### Приклади використання

:: Просте копіювання
robocopy C:\Source D:\Backup

:: Дзеркальне копіювання
robocopy C:\Source D:\Backup /MIR

:: Копіювання з журналом
robocopy C:\Source D:\Backup /LOG:C:\copy_log.txt

:: Багатопотокове копіювання
robocopy C:\Source D:\Backup /MT:8

## Особливості:

Кольорове оформлення для кожного блоку (головне меню, утиліти, DISM, очищення, завершення роботи, Boot Menu).
Перевірка прав адміністратора та автоматичний перезапуск з підвищеними правами.
Логічна структура меню з поверненням у головне меню або виходом.
Повна сумісність із Windows 10/11.

## Як запустити:

Запустіть його від імені адміністратора (через ярлик або контекстне меню).
Використовуйте меню для вибору потрібної дії.

## Примітки
Експорт драйверів йде за адресою D:\MyDrivers
утиліта безпечна для системи.
Деякі функції (наприклад, Mobility Center) доступні лише на ноутбуках.
